# Related Work Draft Paragraphs

Recent diffusion-based decision-making methods frame planning as conditional generation of trajectories rather than direct one-step action prediction. In this view, a denoiser proposes diverse action sequences conditioned on state context, and planning quality emerges from iterative refinement and downstream scoring. Our setup follows this pattern in a discrete domain: a MaskGIT-style denoiser produces candidate Tetris placement sequences `(rotation, x)` conditioned on board geometry and piece identity, then an MPC loop executes only the first action in receding-horizon fashion.

Our implementation is closest to diffusion-MPC formulations that separate candidate generation from candidate evaluation. Specifically, we treat the diffusion model as a proposal distribution and use a reranker to select which candidate to execute. This lets us compare hand-engineered board heuristics against a learned value critic from a pretrained DQN checkpoint while keeping the proposal model fixed. The generate-and-rerank decomposition is useful experimentally because it isolates whether gains come from better search proposals or better value estimation.

MaskGIT-style iterative token refinement is a natural fit for our discrete action representation. The sampler starts from fully masked plans and progressively resolves positions according to model confidence, exposing an explicit compute-quality tradeoff via the number of refinement steps. In Tetris, however, legality constraints are piece-dependent and can invalidate syntactically valid tokens. We therefore include explicit invalid-action handling (penalization or resampling) during planning, which is a critical bridge between generic sequence generation and environment-constrained control.

Adaptive diffusion planning work motivates a next stage beyond static offline training: iteratively improving the proposal distribution with trajectories generated by the current planner. Our current pipeline already supports this progression structurally (dataset collection, denoiser training, MPC evaluation), making it straightforward to test self-improvement loops and alternative teachers (heuristic vs DQN). This places our method within a broader diffusion-for-RL taxonomy: offline generative planning with online critic-guided selection, augmented by domain-specific validity constraints and compute-aware MPC ablations.
